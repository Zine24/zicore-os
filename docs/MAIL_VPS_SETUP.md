# ZICORE Mail Server — Oracle Cloud Free VPS Setup Guide

## Step 1: Create Oracle Cloud Account
1. Go to https://cloud.oracle.com/free
2. Sign up for Always Free tier (no credit card required for Always Free)
3. Create an Ubuntu 22.04 or 24.04 ARM64 instance (1 OCPU, 1GB RAM — free forever)

## Step 2: SSH into your VPS
```bash
ssh ubuntu@YOUR_VPS_IP
```

## Step 3: Run the setup script
```bash
curl -sL https://raw.githubusercontent.com/your-repo/setup-mail-vps.sh | bash
```
Or copy `scripts/setup_mail_vps.sh` to the VPS and run it.

## Step 4: Update Cloudflare DNS
Add these DNS records (I'll do this via API):
- MX: zinemotion.com.mx → mail.zinemotion.com.mx (priority 10)
- A: mail.zinemotion.com.mx → VPS_IP
- TXT (SPF): v=spf1 ip4:VPS_IP include:_spf.mx.cloudflare.net ~all
- TXT (DKIM): (generated by setup script)
- TXT (DMARC): _dmarc.zinemotion.com.mx → v=DMARC1; p=none; rua=mailto:admin@zinemotion.com.mx

## Step 5: Update Cloudflare Tunnel
Point zcs.zicore.space mail routes to the new VPS.

## Step 6: Test
```bash
# From VPS:
echo "Test" | mail -s "ZICORE Mail Test" jilo_tuk@yahoo.com.mx

# From local:
python3 -c "
import smtplib
s = smtplib.SMTP('mail.zinemotion.com.mx', 587)
s.starttls()
s.login('janmad@zinemotion.com.mx', 'password')
s.sendmail('janmad@zinemotion.com.mx', 'jilo_tuk@yahoo.com.mx', 'Subject: Test\n\nHello!')
s.quit()
print('OK')
"
```

## Ports (Oracle Cloud does NOT block these!)
- 25 (SMTP) — outbound
- 587 (SMTP Submission) — inbound
- 465 (SMTPS) — inbound
- 143 (IMAP) — inbound
- 993 (IMAPS) — inbound
- 110 (POP3) — inbound
- 995 (POP3S) — inbound
- 80 (HTTP) — for Let's Encrypt
- 443 (HTTPS) — optional

## Free Tier Limits
- 1 ARM64 instance (4 OCPU, 24GB RAM) or 1 AMD (1 OCPU, 1GB)
- 200GB block storage
- 10GB object storage
- Always Free — never expires
