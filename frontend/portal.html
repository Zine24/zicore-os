MINIMUM

  • Python 3.10 or newer

  • Modern web browser

RECOMMENDED

  • Python 3.11+
  • 8GB+ RAM
  • 10GB+ free disk space
  • Docker Desktop
  • NVIDIA GPU (optional, for Stable Diffusion)

  • Docker 20.10+ (for containers)

  • NVIDIA GPU with 4GB+ VRAM

QUICK START
Up and running in minutes

WINDOWS

# Download and run the installer
zicore_oem_setup.bat

# Or manual install
git clone https://github.com/zinemotion/zicore-system.git
cd zicore-system
pip install -r requirements.txt
python start_all.py

# Open browser
start http://localhost:4000

LINUX

# One-command install
curl -sL https://raw.githubusercontent.com/zinemotion/zicore-system/main/zicore_oem_setup.sh | bash

# Or manual install
git clone https://github.com/zinemotion/zicore-system.git
cd zicore-system
pip3 install -r requirements.txt
python3 start_all.py

# Open browser
xdg-open http://localhost:4000

DOCKER

# Clone and start all services
git clone https://github.com/zinemotion/zicore-system.git
cd zicore-system
docker-compose up -d

# Pull AI models
docker exec zicore-ollama ollama pull tinyllama
docker exec zicore-ollama ollama pull llama3.1:8b

# Open browser
open http://localhost:4000
