Quick Start

# 1. Registrar usuario curl -X POST https://zcs.zicore.space/api/sso/register \ -H "Content-Type: application/json" \ -d '{"email":"user@zinemotion.com.mx","password":"pass123","name":"Mi Nombre"}' # 2. Login y obtener token curl -X POST https://zcs.zicore.space/api/sso/login \ -H "Content-Type: application/json" \ -d '{"email":"user@zinemotion.com.mx","password":"pass123"}' # Response: {"status":"ok","token":"abc123...","user":{...}} # 3. Usar token en requests autenticados curl https://zcs.zicore.space/api/bank/portfolio \ -H "Authorization: Bearer abc123..." # 4. WebSocket ZIO AI ws://zcs.zicore.space/ws/zio?token=abc123...
